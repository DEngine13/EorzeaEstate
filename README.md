# 🏡 Eorzea Estate - FFXIV Real Estate Platform

**Eorzea Estate** is a web-based real estate application dedicated to the housing districts of *Final Fantasy XIV*. 
This project serves as a comprehensive training exercise to master the **Django** framework.

---

## 🚀 Project Goals
The aim is to simulate a complete real estate agency portal for the adventurers of Eorzea:
* **Listing Management:** Dynamic CRUD for houses, Free Company estates, and apartments.
* **Smart Filtering:** Filter properties by district (Mist, Lavender Beds, The Goblet, Shirogane, Empyreum).
* **User Authentication:** Secure dashboards for real estate agents and clients.
* **FFXIV UI Aesthetic:** Custom styling inspired by the game's iconic interface.

## 🛠️ Tech Stack
* **Backend:** Python 3.13 / Django
* **Database:** PostgreSQL
* **Frontend:** HTML5 / CSS3 / Bootstrap
* **Version Control:** Git
